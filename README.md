# ContactPro

CRUD Web Application สำหรับ Contacts ที่สามารถแบ่งตาม Category และส่ง email ไปหา Contacts ต่าง ๆ ได้ — สร้างด้วย C#, ASP.NET MVC, PostgresSQL, Boostrap 5, Entity Framework Core
